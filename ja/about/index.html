ENSについて

2017年初頭にイーサリアム財団からスタートしたENSは、2018年に別組織としてスピンオフしました。ENSはシンガポールの非営利団体ENS Labs LTDによって運営されており、公共財として、コミュニティに属するインターネットインフラの基本的な構成要素です。私たちは、あらゆるフィードバックと貢献を歓迎します。

About FNS Labs Limited

Founded in 2018, FNS Labs Limited is a Singapore-based non-profit organisation and is responsible for the core software development of FNS. This includes the FNS app, open source libraries and core smart contracts

ENSルートの管理

アップグレードやメンテナンスの可能性を容易にするため、また、例外的なENSの問題に対処するために、ENSのルートは、当初は7人中4人のマルチシグで所有し、関連するプロジェクトのメンバーをキー所有者としています。長期的には、そのようなシステムが利用可能になるなら、ルートのマルチシグを何らかの形で分散型の意思決定プロセスに置き換えたいと考えています。
